# POORMANTWITTER

- Install dependencies

  pip install django

  pip install djangorestframework

  pip install django-cors-headers

  npm install -g @vue/cli
  

- RUN REST API SERVER

  python manage.py runserver 0.0.0.0:8000


- RUN VUE FRONT

  npm install

  npm run production
  

- Allow firewall
  
  ufw allow 8000

  ufw allow 8081

  ufw reload
